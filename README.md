# git_connect

### git 설치  --> 기본 값으로 설치

시작버튼에서 Git Bash 열기(실행)

```
git config --global user .name "reha"
```

GitHub 이메일이랑 동일해야함.
```
git config --global user.email "reha01047@gmail.com"
```

정보 확인하기
```
git config --list
```

### GitHub에 코드 업로드 하기

1.초기화

```
git init
```
-.git 이라는 폴더가 생성된다


2.파일 올리기

```
git add .
```


2.히스토리 만들기

```
git commit -m "내가 적고싶은 메세지"  --
```
-메세지에는 한글이 가능하다.
